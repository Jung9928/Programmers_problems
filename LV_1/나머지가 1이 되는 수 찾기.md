## C++ 풀이
```C++
#include <string>
#include <vector>
#include <algorithm>

using namespace std;

int solution(int n) {
    int answer = 0;
    
    
    for(int i=1; i<n; i++)
    {
        if(n % i == 1)
        {
            answer = i;
            break;
        }
    }
    
    return answer;
}
```
